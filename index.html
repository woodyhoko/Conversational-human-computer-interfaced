architecture
A five-stage NLP pipeline.
01

Speech recognition

Google / Bing / Kaldi / DeepSpeech back-ends

→ transcript
02

Intent classification

Hybrid dictionary + ML over 5 intents

→ {READ, SEND, PICK…}
03

Named entity recognition

Bi-LSTM + CRF — 83.7 F1 on the dev set

→ {recipient, content…}
04

Dialogue manager

Slot-filling FSM with clarification + confirm

→ action spec
05

Gmail executor

list / read / send / forward — never without confirm

→ TTS reply
