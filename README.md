# Conversational Human-Computer Interface — Intent Analysis

A research project on **intent analysis for conversational interfaces** — detecting user intent from natural language utterances to improve dialogue systems.

📄 **[Read the Proposal](https://github.com/woodyhoko/Conversational-human-computer-interfaced/blob/main/proposal.pdf)** | 🖼️ **[View the Poster](https://github.com/woodyhoko/Conversational-human-computer-interfaced/blob/main/poster.pdf)**

---

## Overview

As voice and chat interfaces become ubiquitous, accurate intent detection is critical to building responsive conversational systems. This project investigates:

- **Utterance intent classification** — mapping user sentences to predefined intent categories
- Feature engineering from raw text for Mandarin/English conversational data
- Model comparison for sequence classification tasks

The full methodology, dataset description, and experimental results are documented in the research paper (Chinese).
